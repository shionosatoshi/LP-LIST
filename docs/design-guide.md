# マスター農園 デザインガイド

このサイトのトーンマナーとカラーパレットを、あとから見直せるようにまとめた記録です。

## トーンマナー

- スタイリッシュで落ち着いた農園ブランド。
- 余白を広めに取り、情報を詰め込みすぎない。
- 写真は自然光、清潔感、朝の空気感を重視する。
- 文章は短く、活動内容・進捗・面積などの実用情報をすぐ確認できるようにする。
- 予約ページは安心感を優先し、操作に迷わない素直なフォームにする。

## 見た目のルール

- 角丸は基本 `8px`。ボタンや野菜チップなどの小さな選択UIだけ丸くする。
- カードは個別情報のまとまりにだけ使う。ページ全体をカード化しない。
- ヒーローは農園写真を全面に敷き、文字は画像上に直接重ねる。
- 主要CTAは `--green`、補助CTAは透過背景と白枠で控えめにする。
- 進捗・面積などの数値情報は、カード内で大きく読みやすく表示する。

## 使用中の野菜表示

トップページでは、会社名「マスター農園」の下に栽培中の野菜を横並びで表示する。

現在のサンプル:

- トマト
- ナス
- ピーマン
- レタス
- 玉ねぎ
- スナップえんどう

野菜を選択すると、同じページ内で以下を確認できる。

- 現在の進捗
- 作付け面積
- 収穫予定
- 状況説明

## 画像の方向性

- 写真風、リアル、朝の光、清潔な畑。
- 緑、土、淡いゴールドの光を中心にする。
- 文字入り画像は避け、テキストはHTML側で表示する。
- 実在人物、住所、個人情報が写り込む画像は使わない。

## 更新時の注意

- 実在の顧客情報、電話番号、住所、メールアドレス、APIキーはコミットしない。
- 秘密情報やローカル専用ファイルは `.gitignore` に追加する。
- 公開サンプルとして使う場合は、予約フォームが実送信されないことを説明する。
